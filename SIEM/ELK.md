### Elastic Security
**Category:** SIEM  
**Description:**  
Elastic Security provides SIEM capabilities on top of the Elastic Stack (Elasticsearch, Logstash, Kibana).

**Usage:**  
- Ingest and search large volumes of security logs  
- Create detection rules and dashboards  
- Perform threat hunting using KQL or Lucene queries

**Strengths:**  
- Open-source core  
- Highly scalable and customizable  

**Notes:**  
Requires tuning and operational effort to maintain effectively.
